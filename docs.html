📦 安装

VS Code 插件安装

1

打开 VS Code，点击左侧的扩展图标或按 Ctrl+Shift+X

2

在搜索框中输入 "CodeLink"

3

找到 CodeLink 插件，点击 "安装"

4

安装完成后重启 VS Code

JetBrains IDEs 插件安装

1

打开 IntelliJ IDEA / PyCharm 等 JetBrains IDE

2

前往 File → Settings → Plugins

3

搜索 "CodeLink"，点击安装

4

重启 IDE 完成安装

⚙️ 配置

API 密钥配置

首次使用需要配置 API 密钥：

# 打开设置面板
Ctrl+Shift+P (Windows/Linux) 或 Cmd+Shift+P (Mac)

# 输入 "CodeLink: Configure API Key"
# 粘贴您的 API 密钥并保存

代理设置

如果您在企业网络环境中，可能需要配置代理：

{
  "codelink.proxy": {
    "host": "proxy.company.com",
    "port": 8080,
    "auth": {
      "username": "your-username",
      "password": "your-password"
    }
  }
}

🚀 功能介绍

CodeLink 提供三种主要工作模式，每种模式都有其独特的优势和适用场景。

💡 Copilot模式

智能代码补全

开始输入代码，CodeLink 会自动提供智能补全建议：

// 输入函数名，按 Tab 接受建议
function calculateTax(income) {
    // CodeLink 会自动补全函数体
    if (income <= 10000) {
        return income * 0.1;
    } else if (income <= 50000) {
        return 1000 + (income - 10000) * 0.2;
    } else {
        return 9000 + (income - 50000) * 0.3;
    }
}

实时Chat问答

按 Ctrl+Shift+L 打开聊天面板，随时询问编程问题。

🤖 Agent模式

自然语言编程

使用自然语言描述您的需求，Agent 会自动生成代码：

// 在注释中描述需求
// 请创建一个用户登录的 React 组件，包含用户名和密码输入框

// Agent 会自动生成以下代码：
import React, { useState } from 'react';

const LoginComponent = () => {
    const [username, setUsername] = useState('');
    const [password, setPassword] = useState('');
    
    const handleSubmit = (e) => {
        e.preventDefault();
        // 处理登录逻辑
        console.log('Login attempt:', { username, password });
    };
    
    return (
        <form onSubmit={handleSubmit}>
            <input 
                type="text" 
                placeholder="用户名"
                value={username}
                onChange={(e) => setUsername(e.target.value)}
            />
            <input 
                type="password" 
                placeholder="密码"
                value={password}
                onChange={(e) => setPassword(e.target.value)}
            />
            <button type="submit">登录</button>
        </form>
    );
};

export default LoginComponent;

☁️ Remote模式

多任务并行

同时处理多个编程任务，提高开发效率：

# 启动多个并行任务
codelink remote start

# 查看任务状态
codelink remote status

# 任务1: 生成API文档
Task 1: ✅ API documentation generated

# 任务2: 代码重构
Task 2: 🔄 Refactoring in progress...

# 任务3: 单元测试生成
Task 3: ⏳ Waiting in queue

云端同步

您的代码和设置会自动同步到云端，支持跨设备开发。

⌨️ 快捷键

通用快捷键

  • Ctrl+Shift+L 打开聊天面板
  • Ctrl+Shift+A 启动 Agent 模式
  • Ctrl+Shift+R 启动 Remote 模式
  • Tab 接受代码建议

Copilot 模式

  • Ctrl+Space 触发补全
  • Esc 取消建议
  • Alt+→ 部分接受
  • Alt+[ 上一个建议

❓ 常见问题

Q: 插件无法启动怎么办？

A: 请检查是否正确配置了 API 密钥，并确保网络连接正常。如果问题持续，请重启 IDE 并检查插件是否为最新版本。

Q: 代码建议不准确怎么办？

A: 建议提供更多的上下文信息，包括注释和变量名。您也可以通过聊天面板直接询问具体问题。

Q: 如何提高 Agent 模式的效果？

A: 尽量使用清晰、具体的自然语言描述，包含具体的需求和约束条件。避免模糊的表述。

Q: Remote 模式的任务队列满了怎么办？

A: 可以升级到专业版获得更大的任务队列容量，或者等待当前任务完成后再提交新任务。
